# Foodwise

Interactive high-fidelity prototype of Foodwise, a smart kitchen assistant app that reduces food waste through automated pantry tracking, AI-powered recipe suggestions, and smart meal planning.

Built with React + Vite + Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

## Screens

- **Cook** — AI recipe suggestions based on expiring ingredients
- **Pantry** — Automated inventory with expiration tracking
- **Plan** — Weekly meal planner with AI suggestions
- **Shop** — Smart shopping list generated from meal plan
- **Profile** — Savings dashboard and gamification
- **Receipt Scan** — Scan results with estimated expiration dates
- **Weekly Recap** — Quick pantry sync check-in
